---

    title: string.len() 

---

The `string.len` function returns the number of characters in a string.

### Syntax  
```lua
string.len(s)
```  

### Parameters  

#### `s`  
The string to measure.  

### Examples  

#### Basic usage.
```lua
local length = string.len("hello")
print(length)

-- Output: 5
```
